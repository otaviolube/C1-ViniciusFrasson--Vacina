ROTAS PARA TESTAR API Prodution

PESSOA

Cadastrar Pessoa

  • localhost:3001/api/Cadastrarpessoa

    • Encontrar Pessoas

  • localhost:3001/api/ListarVariaspessoas

    • Encontrar uma Pessoa

  • localhost:3001/api/ListarUmapessoa/:cpf

    • Atualizar Pessoa

  • localhost:3001/api/AtualizarPessoa/:cpf

    • Body sample

    { "pessoa_nome":"claudiaS", "data_nascimento": "01/02/1940", "telefone_pessoa": "(27)99721-7421", "grupo_prioritario": false, "endereco_pessoa":" Rua xyz , n 500, bairro abcsdf", "email_pessoa": "mail@mail.com" }

    Deletar Pessoa

  • localhost:3001/api/RemoverPessoa/:cpf

    • UNIDADE

    Cadastrar Unidade

  • localhost:3001/api/CadastrarUnidadeSaude

    • Encontrar Unidade de Saude

  • localhost:3001/api/ListarUnidadesSaude

    • Encontrar uma Unidade Saude

  • localhost:3001/api/ListarUmaUnidadeSaude/:id

    • Atualizar Unidade Saude

  • localhost:3001/api/AtualizarUnidade/:id

    • Body sample

    { "nome_unidade": "TEST UPDATES", "descricao_unidade": "tes desc", "endereco_unidade": "test endereco", "telefone_unidade": "(27)98832-4848", "email_unidade": "mail.com.br", "latlong_unidade": "788855,558888" }

    Deletar Unidade

  • localhost:3001/api/RemoverUnidade/id

    • Agendamento

    Cadastrar Agendamento

  • localhost:3001/api/CadastrarAtendimento/:pessoaid/:unidadeid/

    • Encontrar Atendimentos

  • localhost:3001/api/ListarVariosAtendimentos

    • Encontrar um Atendimento

  • localhost:3001/api/ListarUmAtendimento/:id

    • Atualizar Atendimento

  • localhost:3001/api/AtualizarAtendimento/:dataAtedimento formato (aaaa/mm/dd)

    • Body sample

    "data_hora_agendamento": "2021/07/03", "necessidades_especiais": false, "observacoes_agendamento": "test endereco 55500"

    Deletar Atendimento

  • localhost:3001/api/RemoverAtendimento/:id
